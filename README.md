# LinkedIn Insights Microservice

## Overview
This project is a backend microservice designed to fetch, store, and serve insights of LinkedIn company pages using a Page ID (the last part of a LinkedIn company URL).

The service is designed with a scalable and modular architecture following backend best practices.

---

## Features
- FastAPI-based backend microservice
- Modular project structure
- RESTful API design
- Swagger/OpenAPI documentation
- Environment-based configuration
- MongoDB integration (in progress)

---

## Tech Stack
- Python 3.11
- FastAPI
- MongoDB (Atlas)
- Motor (Async MongoDB Driver)
- Uvicorn
